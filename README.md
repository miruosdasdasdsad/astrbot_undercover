# AstrBot 谁是卧底插件

一个基于 AstrBot 框架开发的谁是卧底游戏插件，支持多人在线参与，自动分配身份和词语，管理游戏流程。

## 功能特点

- 多人在线游戏：支持多个玩家同时参与游戏
- 自动身份分配：根据玩家数量自动计算并分配卧底数量
- **游戏设置**：创建房间后可配置白板开关、词库类别、私聊投票开关
- **分类词库**：内置 100+ 组词语，分为游戏、动漫、影视、美食、日常五大类别
- **多词词组**：每组词可包含 2 个以上词语，游戏随机取 2 个，重开不重样
- **特殊模式**：极小概率触发 1 卧底 + 全白板的神秘对局
- 完整游戏流程：创建房间 → 加入游戏 → 开始游戏 → 轮流发言 → 投票淘汰 → 判定胜负
- 词语库管理：支持玩家自定义添加词语
- 简短指令：使用 `/uc` 作为指令前缀，输入更快捷

## 安装方法

1. 将插件目录复制到 AstrBot 的 `data/plugins/` 目录下
2. 重启 AstrBot 即可自动加载插件

## 使用说明

### 游戏设置（新增 v1.4.0）

创建房间后，房主可使用 `/uc config` 调整游戏设置：

```
/uc config                        # 查看当前设置
/uc config whiteboard on|off      # 开关白板（默认开）
/uc config category <类别名>      # 选择词库（默认全部）
/uc config vote private|public    # 私聊投票开关（默认开）
/uc categories                    # 查看所有可用词库类别
```

### 指令列表

| 指令 | 功能 | 权限 |
|------|------|------|
| `/uc help` | 查看帮助信息 | 所有人 |
| `/uc create` | 创建游戏房间 | 所有人 |
| `/uc join <房间号>` | 加入游戏房间 | 所有人 |
| `/uc config [设置] [值]` | 查看/修改游戏设置 | 房主（开始前） |
| `/uc categories` | 查看可用词库类别 | 所有人 |
| `/uc start` | 开始游戏 | 房主 |
| `/uc leave` | 离开当前房间 | 所有人 |
| `/uc say <内容>` | 游戏中发言 | 游戏中玩家 |
| `/uc vote <玩家>` | 游戏中投票 | 游戏中玩家 |
| `/uc guess <词语>` | 白板猜词 | 白板玩家 |
| `/uc end` | 结束游戏 | 房主 |
| `/uc add <词语1> <词语2> ...` | 添加词语 | 所有人 |
| `/uc word` | 查看我的词语 | 所有人 |
| `/uc list` | 查看游戏列表 | 所有人 |

## 游戏规则

### 普通模式

1. **身份分配**：
   - 8人及以下：1个卧底
   - 8人以上：2个卧底
   - 5人及以上（白板开启时）：1个白板（无词语）

2. **游戏流程**：
   - 平民获得相同词语，卧底获得相似但不同的词语
   - 玩家轮流发言描述自己的词语
   - 发言结束后投票淘汰可疑玩家
   - 重复直到一方胜利

3. **胜负判定**：
   - 平民胜利：所有卧底和白板被淘汰
   - 卧底胜利：卧底数量 ≥ 平民数量
   - 白板被票出后可猜词，猜对即获胜

### 特殊模式（隐藏）

白板开启时，有小概率触发特殊对局：只有 1 人知道词语（卧底），其余全部为白板。白板可随时猜词，猜对即获胜。不提示玩家进入此模式，全凭玩家自己察觉。
